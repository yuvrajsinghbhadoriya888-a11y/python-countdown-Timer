# Python Countdown Timer

A simple countdown timer built using Python.

## Features

* Takes countdown duration as user input.
* Displays time in MM:SS format.
* Updates the timer every second.
* Plays a beep sound during the countdown.
* Displays "Time's up!" when the countdown reaches zero.

## Technologies Used

* Python
* time module
* winsound module (Windows)

## How It Works

1. User enters the countdown duration in seconds.
2. The program converts the total seconds into minutes and seconds using `divmod()`.
3. The timer updates every second.
4. When the timer reaches zero, an alert sound is played and the program displays a completion message.
